# PyCraft - Public

This repository is the public space for issues and the roadmap for PyCraft.

It does not contain the game source code. The source code is intentionally kept in a private
repository. 

Use this public repository to report bugs, request new features, and follow planned work.

## Purpose

Use this repository for:

- bug reports
- feature requests
- public roadmap discussion

Please do not use this repository for general support questions, private information, or source-code requests.

## Where To Find The Software

Download PyCraft or learn more [here](https://simplay.github.io/pycraft/).

## How To Report A Bug

Before opening a new issue, search the existing issues first. If another issue
already describes the same problem, add your details there instead of opening a
duplicate.

When reporting a bug, include:

- PyCraft version
- system specifications: operating system, CPU, RAM, and GPU model
- expected behavior: what you expected to happen
- actual behavior: what actually happened
- steps to reproduce: A sequence of actions that triggered the problem
- coordinates: for visual, terrain, chunk loading, or world generation bugs,
  include your in-game X, Y, Z coordinates if available (F2)
- attachments: screenshots, videos, logs, or crash messages when available

If the game crashes, attach `game.log` from the PyCraft log directory:

- Windows: `%APPDATA%\pycraft\logs\game.log`
- macOS: `~/Library/Application Support/pycraft/logs/game.log`
- Linux: `${XDG_DATA_HOME:-~/.local/share}/pycraft/logs/game.log`

Please keep each issue focused on one problem. This makes bugs easier to track,
test, and close.

## Feature Requests

Feature requests are welcome when they describe a concrete player-facing
improvement. Explain what you want to do in the game, why it matters, and any
examples from similar games if they help clarify the idea.

## Roadmap

The public roadmap tracks larger planned work and known priorities. Roadmap
items may change as development priorities, technical constraints, and player
feedback change.
